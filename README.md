# cube3D
A custom 3D raycasting engine developed in C, inspired by the classic Wolfenstein 3D.

## Features
* **Raycasting Engine:** Implemented from scratch to render a 3D perspective from a 2D map.
* **Collision Detection:** Developed efficient logic to handle player movement and wall interactions.
* **Rendering:** Custom graphics handling for textures, sprites, and floor/ceiling colors.
* **Input Handling:** Smooth keyboard event processing for player movement and camera rotation.

## Build
```bash
git clone <repository-url>
cd cub3D
make
./cub3D maps/map.cub
